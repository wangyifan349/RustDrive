# RustDrive ☁️

RustDrive 是一个使用 Rust 编写的轻量级自托管云盘项目。它基于 Axum、Tokio 和 SQLite 构建，前端页面直接内嵌在后端程序中，项目结构简单，部署成本低，适合个人服务器、家庭 NAS、局域网文件管理、小型团队共享文件以及 Rust Web 项目学习。

RustDrive 不是一个只能上传和下载文件的简单示例，而是一个功能比较完整的云盘系统。它支持用户账户、文件上传下载、文件夹管理、文件移动、文件重命名、文件删除、多选批量操作、7z 批量下载、公开分享、分享目录浏览、分享文件下载、分享内容批量下载、在线预览、在线视频播放、在线音频播放、文本在线查看和文本在线编辑。文件数据默认保存在本地目录中，元数据使用 SQLite 保存，不需要额外部署 MySQL、PostgreSQL、Redis 或对象存储服务。

## 功能特性 ✨

RustDrive 提供完整的文件和目录管理能力。用户可以注册、登录、退出，登录后可以创建文件夹、上传文件、重命名文件或文件夹、移动文件或文件夹、删除文件或文件夹，并支持多选操作。对于多个文件或文件夹，可以批量下载为 7z 压缩包，方便一次性导出数据。

项目支持丰富的在线预览能力。图片、PDF、文本等常见文件可以直接在网页中查看；视频文件支持在线播放；音频文件支持在线播放；文本文件支持在线查看，并且对于支持编辑的文本类型，可以直接在网页中修改和保存。

RustDrive 拥有完整的分享功能。用户可以为文件或文件夹创建公开分享链接，也可以取消分享。被分享的文件可以直接下载或在线预览；被分享的文件夹可以在分享页面继续浏览子目录、查看文件列表、下载单个文件、预览支持的文件，也可以批量下载分享内容。分享访问是只读的，适合临时向他人提供文件或目录访问能力。

在文件处理方面，项目支持本地对象存储、SQLite 元数据管理、文件大小记录、MIME 类型识别、同名文件处理、文本编码检测、Range 请求支持和 7z 压缩下载。上传、移动、删除、分享、下载等关键操作会在后端终端输出必要的运行信息，方便部署后观察程序状态和排查问题。

## 主要能力概览

- 支持用户注册、登录、退出
- Cookie 会话管理
- 文件上传、下载、删除、重命名和移动
- 文件夹创建、删除、重命名和移动
- 多选文件和文件夹批量操作
- 多文件、多目录批量下载为 7z
- 图片、音频、文本文件在线预览，支持分享并预览。
- 视频文件在线播放
- 音频文件在线播放
- 文本文件在线查看
- 文本文件在线编辑和保存
- 文件公开分享
- 文件夹公开分享
- 分享目录在线浏览
- 分享文件在线预览
- 分享文件下载
- 分享内容批量下载
- 本地 SQLite 数据库存储
- 本地文件对象存储
- 自动检测证书，可用时启用 HTTPS，不可用时使用 HTTP
- 后端终端输出关键运行信息

## 技术栈 🦀

Rust、Axum、Tokio、SQLx、SQLite、Serde、sevenz-rust、chardetng、sha2。

## 获取项目

```bash
git clone https://github.com/wangyifan349/RustDrive.git
cd RustDrive
cargo build --release
```

## 编译

```bash
cargo build --release
```

## 运行 🚀

```bash
cargo run --release
```

默认访问地址：`http://127.0.0.1:3000`。如果当前目录下存在可用证书，程序会自动尝试使用 HTTPS；
如果没有证书或证书不可用，会自动使用 HTTP，不影响正常运行。

## 数据存储

程序运行后会在本地创建数据目录，用于保存 SQLite 数据库、上传文件对象和临时压缩文件。
正式使用时请定期备份数据目录，避免磁盘故障或误删除导致数据丢失。

## 适用场景

RustDrive 适合个人自托管云盘、家庭服务器文件管理、局域网文件共享、小型团队临时文件分发、Rust Web 开发学习和轻量级私有文件服务。如果你需要一个结构简单、依赖少、容易编译和部署的 Rust 云盘项目，RustDrive 可以作为一个完整的基础版本继续扩展。
